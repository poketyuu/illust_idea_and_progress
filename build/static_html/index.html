アイデアを整理して
完成まで進捗管理

いつでもどこでも思いついたアイデアをその場でメモ。各アイデアについて現在の進捗状況を記録し、 計画的かつ能率的に創作を進めましょう。タグの追加や締切の設定を行うことで、アイデアが増えても 簡単に作業を行うアイデアを確認することができます。モバイル、デスクトップいずれの端末からも ご利用可能です。
より良い進捗管理でより良い創作活動を！

How to use?

進捗設定

[メニュー]→[マイ進捗]より自分に好みに進捗状況を追加、編集しましょう。

アイデア追加

[新規アイデア]よりいつでもどこでも新しいアイデアを追加しましょう。 タグの追加や締切の設定はここで行える他、アイデア作成後に編集することも可能です。

進捗更新

進捗があった際にはトップ画面および各アイデアの個別画面のドロップダウンメニューより進捗状況の変更を行いましょう。 変更ボタンを押すと進捗状況が更新されます。完成目指して頑張りましょう！

アイデア一覧

[アイデア一覧]では完成したアイデアも含めた全てにアイデアが確認できる他、タグやキーワードを用いた検索ができます。 アイデアの整理等にお役立てください。
