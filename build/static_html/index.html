アイデアを整理して
完成まで進捗管理

いつでもどこでも思いついたアイデアをその場でメモ。各アイデアについて現在の進捗状況を記録し、 計画的かつ能率的に創作を進めましょう。タグの追加や締切の設定を行うことで、アイデアが増えても 簡単に作業を行うアイデアを確認することができます。モバイル、デスクトップいずれの端末からも ご利用可能です。より良い進捗管理でより良い創作活動を！

How to use?

1. メニュー → マイ進捗より進捗状況の一覧を編集。自分好みに進捗状況を追加、削除して 進捗順に並び替えましょう。

2. 新規アイデアより、新しいアイデアを追加しましょう。タグの追加や締切の設定等はここで行える他、 アイデア作成後に編集することも可能です。追加されたアイデアは進捗順、締切順、また作成が古い順にトップ 画面に表示されます。

3. 各アイデアに進捗があった際にはトップ画面および各アイデアの個別画面のドロップダウンメニューより 進捗状況の変更を行いましょう。変更ボタンを押すと進捗状況が更新されます。完成目指して頑張りましょう！

4. アイデア一覧画面では完成したアイデアも含めて全てのアイデアが確認できる他、タグやキーワードを 用いた検索を行うことが可能です。上手にアイデアを整理して進捗上手になりましょう。
